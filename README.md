# Übungsprojekte

Dieses Repository dokumentiert meine kontinuierliche Weiterbildung als Softwareentwickler.  
Schwerpunkte: **Relationale Datenbanken & SQL** sowie **Python-Grundlagen und Algorithmen**.

---

## Inhalt

### 📂 relational-databases
Übungen zu relationalen Datenbanken auf Basis der freeCodeCamp-Kursinhalte.

Behandelte Themen:
- Datenbankdesign und Normalisierung
- SQL-Abfragen: SELECT, JOIN, Subqueries, Aggregatfunktionen
- Tabellenbeziehungen und Fremdschlüssel
- Praktische Übungsdatenbanken mit PostgreSQL

→ Details im [README des Ordners](./relational-databases/README.md)

### 📂 Python
Grundlagen und Konzepte der Python-Programmierung.

Behandelte Themen:
- Kontrollstrukturen, Funktionen, Fehlerbehandlung
- Datenstrukturen (Listen, Dictionaries, Sets)
- Skriptbasierte Automatisierung

---

## Warum dieses Repository?

Neben meiner täglichen Arbeit als Anwendungsentwickler nutze ich strukturierte Lernprojekte,  
um mein Wissen zu vertiefen und neue Konzepte gezielt auszuprobieren.  
Alle hier enthaltenen Projekte sind eigenständig erarbeitet.
